# ineqfun_data

Normalized number of total cells per sample

## Usage

``` r
ineqfun_data(pars, total_cells)
```

## Arguments

- pars:

  A numeric vector of length equal to no. of samples, representing their
  corresponding normalization factors

- total_cells:

  An integer vector representing the total number of cells per sample

## Value

A numeric vector representing the normalized number of total cells per
sample
